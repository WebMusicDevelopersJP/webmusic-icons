webmusic-icons


Scalable icons for Web Music Applications

Overview

'webmusic-icons' is a icon-set for Web Music Applications.

  • Scalable : Icons are designed using SVG.
  • Supported Format : [Polymer] iron-iconset-svg (iron-icon Tag). Also available image file download as PNG / SVG

Usage

1. Use as PNG or SVG file

If you want to use these icons just as simple PNG or SVG file, click the icon in the list and download the PNG/SVG image file (This function is available for Chrome / Firefox / Edge only).

Following steps are for 'iron-icon' tag usage.

2. Install 'webmusic-icons.html' and depended files.

webmusic-icons is a Polymer's custom icons with iron-iconset-svg. Though it need some preparation to use, icons can be used with simple <iron-icon> tag.

If you use bower, use the command below. It automatically solve the dependency and place files to 'bower_components' folder : 

  > bower install --save webmusic-icons
Otherwise, download 'webmusic-icons.html' from this link :
webmusic-icons.html

And follow links below for manually install other required files :
polymer 1.0
PolymerElements/iron-icon
PolymerElements/iron-iconset-svg

All files should be placed appropriate locations.

3. Load required libraries to your HTML

Add script & link tags for polymer, iron-icon, iron-iconset-svg and webmusic-icons to your HTML. In this description, all of the components assumes be placed under 'bower_components' folder by bower command. 

<!doctype html>
<html>
.
.
<script src="bower_components/webcomponentsjs/webcomponents.min.js"></script>
<link rel="import" href="bower_components/iron-icon/iron-icon.html">
<link rel="import" href="bower_components/iron-iconset-svg/iron-iconset-svg.html">
<link rel="import" href="bower_components/webmusic-icons/webmusic-icons.html">
.
.
.
</html>

4. Use Icon

Insert a tag to your HTML like below. 'iconName' specifies the type of icon : 
<iron-icon icon="webmusic-icons:iconName"></iron-icon>

For example...

<iron-icon icon="webmusic-icons:midi"></iron-icon>

Add style-attribute if you want to change size or color :

<iron-icon icon="webmusic-icons:midi" style="width:100px;height:100px;color:magenta"></iron-icon>
